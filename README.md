**Neural Guardian** to zaawansowany, multimodalny terminal operacyjny zbudowany w czystym JavaScript, integrujący biometrię twarzy AI, zaawansowane zarządzanie mediami i interaktywną konsolę dowodzenia. Projekt łączy futurystyczny interfejs typu „Hacker UI” z realną użytecznością biznesową.

## 🚀 Kluczowe Funkcjonalności

* **🧬 Biometric Face Tracking:** Integracja z Google MediaPipe Face Mesh do śledzenia 468 punktów twarzy w czasie rzeczywistym.
* **🎙️ AI Voice Feedback:** System komunikacji głosowej (Text-to-Speech) informujący o statusie systemu i autoryzacji.
* **⌨️ Command-Line Interface (CLI):** W pełni funkcjonalna konsola do sterowania odtwarzaczem i zasobami sieciowymi za pomocą komend.
* **📺 Global Streaming Hub:** Zintegrowany agregator ponad 100 serwisów VOD i kanałów LIVE posortowany alfabetycznie (wpisz literę, np. `a` i naciśnij enter).
* **💾 Multi-Format Capture:** Moduł pozwalający na eksport skanów biometrycznych do formatów JPG oraz PNG.
* **🎵 Synchronized Media Engine:** Zaawansowany system audio-video. Aby zadziałała muzyka, najpierw wpisz `muzyka` i wciśnij play. Video otwiera się po komendzie `open_window`, a siatka twarzy po komendzie `skan`.

## 🛠️ Stack Techniczny

- **Frontend:** HTML5, CSS3 (Custom Matrix Filters), Tailwind CSS
- **AI/Vision:** MediaPipe Face Mesh, MediaPipe Camera Utils
- **Logic:** Vanilla JavaScript (ES6+)
- **API Integration:** Jitsi Meet (Secure Comms), SpeechSynthesis API

## 📸 Interfejs Użytkownika

Program został zaprojektowany w estetyce *Retro-Futurism*. Wykorzystuje autorskie filtry dynamiczne, animowane skanery biometryczne oraz holograficzne wizualizacje generowane przez algorytmy trygonometryczne (sin/cos).

## 📥 Instalacja i Uruchomienie

Projekt nie wymaga kompilacji (Zero-Config).

1. Sklonuj repozytorium:
   ```bash
   git clone https://github.com
   ```
2. Otwórz plik `index.html` w dowolnej nowoczesnej przeglądarce.
3. Zezwól na dostęp do kamery dla modułu biometrycznego.
